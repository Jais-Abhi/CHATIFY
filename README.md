## üß† Introduction

**Chatify** is a modern, full-stack **real-time chatting web application** built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js) and **Socket.io**.  
It provides a smooth and secure messaging experience ‚Äî whether for one-to-one or group conversations.

With its **clean UI**, **instant communication**, and **powerful backend**, Chatify gives users a social experience similar to modern chat apps ‚Äî but completely open-source and customizable!

## üéØ Project Goals

- To create a **real-time chat app** that supports live message exchange without page reloads.  
- To build a **secure and scalable** backend using Node.js, Express, and MongoDB.  
- To provide an **interactive and responsive UI** using React and Tailwind CSS.  
- To integrate **Cloudinary** for image storage and **JWT authentication** for user security.
